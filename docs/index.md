# AegisAV Documentation

This site covers how to run, tune, and extend AegisAV. It focuses on:

- Configuration and tunables
- Agent decision pipeline
- Scenario execution and simulation
- AirSim integration and overlay behavior
- Operations and troubleshooting

## Quick Start

1) Install and configure dependencies.
2) Start the server.
3) Open the dashboard and overlay.
4) Run a scenario or post vehicle state to `/state`.

See `guide/getting-started.md` for full steps.
